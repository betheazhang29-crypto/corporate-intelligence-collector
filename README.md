# 企业情报采集专家

专业的企业情报采集技能，从10+公开渠道采集企业全方位信息，支持3种工作模式（快速/标准/深度），并行采集提高效率。

## 功能特点

- **10+数据源**：企查查、天眼查、国家企业信用信息公示系统、上市公司年报、行业研究报告等
- **4种工具策略**：Browser（浏览器自动化）、Search（深度搜索）、File（文档解析）、Shell（批量处理）
- **3种工作模式**：快速（10分钟）、标准（30分钟）、深度（60分钟）
- **并行采集**：同时启动多个采集任务，提高效率
- **标准化输出**：JSON数据 + 截图证据 + 原始文档

## 使用方法

```
使用 corporate-intelligence-collector 技能采集【某某公司】的企业信息
```

## 输出

- `intelligence_data.json`：标准化的企业情报数据
- `screenshots/`：截图证据
- `documents/`：原始文档

## 独立使用

本技能可以独立使用，也可以作为完整尽调流程的一部分。
